minDelta number <optional>

poller null | object | Poller <optional>
 The poller to use or the options for the poller. If the the value is of instance Poller it is used as-is; otherwise if it's a plain dictionary it is used as config object for the Poller instance which will get created. If poller is not set or set to null no poller is associated with this progressbar and no binding of events or callbacks will be done. If no poller is defined or this is set to null, no polling will occur. For options see the constructor options of Poller.
